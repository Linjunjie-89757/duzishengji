# 自动化测试平台项目说明

这个目录用于给项目协作方查看当前最新项目结构，包含新重建前端和旧项目后端。

## 目录结构

- `frontend/`：新重建前端，技术栈为 Vue 3、Vite、TypeScript、Element Plus。
- `backend/`：旧项目后端，来源于 `D:\Project\auto\server`，技术栈为 Spring Boot、Java、Maven。

## 启动后端

```powershell
cd backend
mvn spring-boot:run
```

如果本机 Java 或 Maven 未配置到环境变量，需要先配置对应版本后再启动。

## 启动前端

```powershell
cd frontend
npm install
npm run dev
```

前端本地开发环境建议使用：

```env
VITE_API_BASE_URL=http://localhost:8080/api
```

可以在 `frontend/.env.local` 中配置以上地址。

## 交接说明

本次整理只复制源码、配置和必要文档，已排除依赖、构建产物、运行日志和本地临时数据，例如：

- `node_modules`
- `dist`
- `target`
- `logs`
- 本地 H2 数据文件

如需完整运行，请分别在 `frontend` 和 `backend` 目录安装依赖并启动服务。
