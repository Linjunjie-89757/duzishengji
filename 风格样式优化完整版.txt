自动化测试平台完整视觉规范（优化完整版｜含新增响应式 / 交互动效 / 图标规范 + 表格尺寸修正）
风格定位
轻量化 B 端企业测试平台极简商务风，基于 Element Plus 原生组件体系，低视觉负荷、高信息密度，适配测试人员长时间办公；统一变量、统一交互、统一布局，多人协作零视觉分歧，前端工程化无死角落地。
一、整体色彩体系
1. 主品牌色
主蓝 #409EFF
使用场景：激活菜单、Tab 下划线、高亮按钮、链接、分页激活、输入框聚焦边框、可点击文字
特点：低饱和科技蓝，长时间浏览不刺眼
2. 侧边栏深色基底
侧边栏背景 #232935（非纯黑，降低压迫感）
侧边文字：#ffffff；激活菜单文字复用主蓝色
3. 三层页面中性底色
全局页面背景：#F5F7FA（页面空白区域）
卡片容器底色：#FFFFFF（树形、表格、筛选、编辑器统一承载）
空白 / 占位区域：#FAFAFA（请求体、无数据提示框）
4. 功能状态色（测试业务专用）
表格
色值	业务场景
#67C23A	执行成功、通过标签
#E6A23C	执行中、阻塞、P1 优先级
#F56C6C	失败、P0 高优、删除危险操作
#909399	次要文字、备注、失效提示
5. 分割边框色
常规边框：#E4E7ED
栏分割线：#EBEEF5
二、布局结构规范
全局固定三栏骨架
左侧导航（固定宽度）+ 顶部全局 Header + 弹性主内容区
左侧侧边栏
固定宽度：220px；永久悬浮滚动不消失；顶部产品名称，一级菜单 + 可折叠二级子菜单；文字单行截断，不溢出容器。
顶部 Header
固定高度：56px；纯白卡片容器；
左侧：环境 / 工作空间下拉；右侧：刷新、新增模块按钮。
主内容自适应
所有页面基础最小可视宽度：1280px
浏览器宽度 ≥1280px：布局自适应拉伸；
浏览器宽度 ＜1280px：页面出现横向滚动条，禁止挤压、变形、换行。
页面通用从上至下分层模板
页面标题区 → 顶部分类 Tab → 筛选操作卡片 → 核心数据 / 功能面板 → 分页底部
测试专用双栏模板（接口 / 用例页面）
左侧树形目录（固定 260px）+ 右侧编辑 / 表格弹性区域
高度统一 calc(100vh - 260px)，内部单独滚动，不整页滚动。
三、卡片、圆角、阴影规范
圆角分级统一变量
大功能卡片（树 / 表格 / 编辑器）：$radius-md:8px
按钮、输入框、Tag、小空白框：$radius-sm:4px
数据统计卡片：$radius-lg:12px
阴影规范
全局统一轻量阴影：0 2px 12px rgba(0,0,0,0.06)
禁止厚重深色阴影，仅做分层区分，无悬浮加重阴影。
容器规则
所有树形、表格、筛选、编辑模块必须包裹白色卡片，不直接裸露页面底色。
四、字体与间距排版系统
字体层级（无特殊艺术字体，系统无衬线）
数据大数字：36px 加粗（统计卡片指标）
页面 / 模块标题：16px 600 粗体
正文、表格、按钮标准字：14px
辅助备注、英文状态：12px 浅灰弱化
标准化间距变量
$spacing-xl:32px 页面左右边距、侧边栏顶部留白
$spacing-lg:24px 模块垂直大间隔
$spacing-md:16px 卡片内边距、元素横向间隔（高频）
$spacing-sm:8px 按钮组、输入框缝隙
$spacing-xs:4px 文字与标签微小间隙
文字溢出规则
菜单、树节点、用例名称全部单行：
white-space: nowrap; overflow: hidden; text-overflow: ellipsis;
超长文本悬浮展示完整 Tooltip
五、组件视觉统一规范
1. 按钮分级
主填充按钮：#409EFF 白底字（新建、查询、发送）
常规边框按钮：白底灰边灰字（导入、重置、Curl）
文字按钮：无边无背景，主蓝色小字（表格编辑 / 详情）
危险文字按钮：红色（删除）
Tab 内小型操作按钮统一 size="small"
2. Tag 状态标签
统一 4px 圆角，紧凑小号；
优先级：P0 红 / P1 橙 / P2 灰；
执行状态：成功绿 / 阻塞橙 / 失败红。
3. 树形组件
白色卡片承载，默认全部展开；顶部搜索 + 操作按钮；超长节点悬浮 Tooltip 展示全称。
4. 表格（修正行高）
统一行高：48px（原 56px 留白过大，高密度测试数据优化）
表头背景 #FAFAFA；垂直居中；操作列固定右侧；
底部分页左右两端：左侧总条数，右侧分页组件。
5. Tab 标签
顶部大分类 Tab、编辑器内嵌 Tab 统一蓝色下划线激活，无花哨样式。
6. 输入 & 下拉
浅灰边框，聚焦自动变为主题蓝色；多筛选控件自动换行自适应。
六、空状态 & 异常 & 加载规范（新增板块）
1. 空页面统一样式
容器底色 #FAFAFA，居中布局
大空场景（表格 / 树无数据）：56px 灰色装饰图标 + 14px 灰色提示文字，底部可放置操作按钮；
小型空白块（接口请求体 / 响应）：仅居中文字，不放置大图标。
2. 异常状态（接口报错、加载失败）
底色不变，图标改为红色失败图标，文字红色，下方增加「刷新」按钮。
3. 全局加载方案
全部使用 Element Plus 内置 <el-spin>，禁止自定义骨架屏；
表格、树形、编辑器加载时全屏遮罩。
七、图标体系规范（新增板块）
图标库约束：仅使用 Element Plus 内置图标，不引入 Iconify、阿里图标等第三方库，减少打包与兼容问题；
尺寸分级
侧边菜单图标：18px
按钮内操作图标：14px
空状态装饰图标：56px
表格更多小图标：14px
颜色规则
通用图标继承父文字色 color: currentColor；
成功 / 失败状态图标单独使用功能色，禁止硬编码色值。
八、动效与交互反馈（新增板块）
全局统一过渡动画：all 0.25s ease，所有组件复用
侧边栏菜单 hover 背景：rgba(255,255,0.08)
表格行 hover 背景：#F5F7FA
按钮 hover 色
主按钮 #409EFF → #66B1FF
危险按钮 #F56C6C → #F78989
普通边框按钮 hover：浅灰底色
弹窗、抽屉、侧边栏折叠动画统一 0.25s 淡入淡出；
所有点击元素 hover 必须有视觉区分，禁止无反馈静态组件。
